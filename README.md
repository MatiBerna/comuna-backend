# Nombre del proyecto: comuna-backend

## Instalación
1. Clona este repositorio.
2. Ejecuta el siguiente comando para instalar las dependencias:
```bash
npm install
```

## Uso
### Modo de Desarrollo
Ejecuta el siguiente comando para iniciar la aplicación en modo de desarrollo:
```bash
npm run start:dev
```
